# CuakForesta :duck:
En este juego, el objetivo es recolectar recursos para reforestar el 70% de los árboles. Además, habrá enemigos que intentarán evitar que se cumpla el objetivo.

## Cómo jugar :video_game:
1. Se inicia el nivel.
2. Se debe recolectar los recursos y acercarse a la zona deseada para reforestar, esto permitirá que los botones se activen y el contorno rojo permitirá saber qué recurso falta.
3. Se debe esquivar o disparar a los enemigos mediante el mouse. Además, estos no pueden sobrevivir por cierto tiempo fuera de su zona contaminante. 
4. Al plantar un árbol, se activará su poder correspondiente, que se podrá usar cada cierto intervalo.
5. Reforesta el 70% para ganar y desbloquear el siguiente nivel.
